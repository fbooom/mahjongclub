# Onboarding Checklist — Mahjong Club

Use this checklist to get fully set up your first time. Most steps take less than a minute.

---

## Getting Started

- [ ] **Create your account** — sign up with email or Google on the sign-in screen
- [ ] **Set your display name** — Account tab → Edit Profile → update your name
- [ ] **Choose an avatar** — pick an emoji that feels like you
- [ ] **Set your skill level** — Beginner, Intermediate, or Advanced (helps balance seating plans)
- [ ] **Pick a theme** *(optional)* — Account tab → Theme → choose Flowers 🌸, Bam Bird 🐦, Dragons 🐉, or Tiles 💠

---

## Join Your Group

- [ ] **Get a join code or invite link** from your group host or a member
- [ ] **Join the group** — Groups tab → Join Group → enter the code
- [ ] **Say hello in the group chat** — Groups tab → your group → Chat

---

## Enable Notifications

- [ ] **Turn on notifications in the app** — Account tab → Notifications → toggle on
- [ ] **Allow notifications on your device** — tap Allow when the system permission prompt appears
- [ ] **iPhone users:** use Safari (not Chrome) for the best notification experience

---

## Respond to Your First Game

- [ ] **Check your invitations** — Games tab shows any upcoming games you've been invited to
- [ ] **RSVP** — tap a game and choose Going, Maybe, or Can't Go
- [ ] **Add the game to your calendar** *(optional)* — game detail → Add to Calendar

---

## Host Checklist *(only if you're organizing games)*

- [ ] **Create your group** — Groups tab → + → name it and choose an emoji
- [ ] **Share your group join code** with your players
- [ ] **Schedule your first game** — Games tab → + → fill in title, date, time, and location
- [ ] **Invite your players** — game detail → Invite → share via SMS, link, or QR code
- [ ] **Set up seating** before game day — game detail → Seating → Generate → publish when ready
- [ ] **Record the winner** after the game — game detail → Record Winner

---

## You're All Set

Once you've checked everything above, you're ready to play. Need help? See the [User Guide](user-guide.md) for a full walkthrough of every feature, or the [FAQ](faq.md) for quick answers to common questions.
